# V11 SMPLX 映射配置优化报告

**报告时间**: 2026年2月4日 14:47  
**项目**: GMR - V11 SMPLX 关节映射精确化  
**专家**: 资深强化学习人形机器人专家顾问  

## 1. 任务概述

本次任务旨在精确修改 `smplx_to_v11.json` 配置文件，优化SMPLX人体模型到V11机器人的关节映射关系。原始配置存在以下问题：

1. 关节映射不完整
2. 四元数对齐存在较大偏差（部分关节差异高达90°）
3. 权重设置不合理（特别是末端执行器位置权重为0）
4. 缺少V11特有关节的映射

## 2. 分析方法

### 2.1 关节结构分析

**V11机器人关节层次（从URDF分析）：**

- **腿部关节链**（每侧6个关节）：
  - 髋关节：pitch, roll, yaw
  - 膝关节：pitch
  - 踝关节：pitch, roll

- **手臂关节链**（每侧7个关节）：
  - 肩关节：pitch, roll, yaw
  - 肘关节：pitch
  - 腕关节：roll, pitch, yaw

**SMPLX关键关节（15个）：**
pelvis, left_hip, right_hip, spine3, head, left_knee, right_knee, left_foot, right_foot, left_shoulder, right_shoulder, left_elbow, right_elbow, left_wrist, right_wrist

### 2.2 映射完整性分析

通过对比分析发现：

1. **V11配置有15个关节映射**
2. **G1配置（参考）有14个关节映射**
3. **当前映射覆盖率**：
   - 左腿：3/6 (50%)
   - 右腿：3/6 (50%) 
   - 左臂：3/7 (43%)
   - 右臂：3/7 (43%)

## 3. 问题识别

### 3.1 四元数对齐差异

通过计算与G1配置的四元数角度差异，发现以下严重问题：

| 关节名称 | 与G1差异角度 | 严重程度 |
|---------|-------------|----------|
| left_elbow_link | 73.7° | 严重 |
| left_wrist_yaw_link | 73.7° | 严重 |
| right_elbow_link | 91.0° | 严重 |
| right_wrist_yaw_link | 90.0° | 严重 |
| left_hip_roll_link | 15.8° | 中等 |
| right_hip_roll_link | 15.8° | 中等 |

### 3.2 权重设置问题

末端执行器位置权重过低：
- `left_wrist_yaw_link`: 位置权重为0（应≥50）
- `right_wrist_yaw_link`: 位置权重为0（应≥50）

## 4. 优化方案

### 4.1 配置文件改进点

已生成优化配置文件 `smplx_to_v11_improved.json`，主要改进包括：

1. **启用 ik_match_table2**: 获得更好的腿部控制
2. **调整位置权重**:
   - 脚踝关节：100（保持）
   - 手腕关节：0 → 80（大幅提高）
   - 腿部关节：适当调整（hip: 20-30, knee: 10）
3. **优化四元数对齐**: 基于G1配置修正
4. **增加精细映射**: 更详细的关节链对应关系
